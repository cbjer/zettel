# Sum of Subsets Definition

The sum of subsets of $V$ denoted $U_1 + \ldots + U_m$ is the set of all possible sums of elements in $U_1, \ldots U_m$.
Ie

$$
U_1 + \ldots + U_m = \{ u_1 + \ldots + u_m : u_1 \in U_1, \ldots, u_m \in U_m \}
$$
