# Basic Properties of the Adjoint

The [adjoint](202102161843_adjointDefinition) has the following properties:
- $(S + T)^* = S^* + T^*$
- $(\lambda T)^* = \overline{\lambda} T^*$
- $(T^*)^* = T$
- $I^* = I$
- $(S T)^* = T^* S^*$
