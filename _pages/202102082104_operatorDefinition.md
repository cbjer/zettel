# Operator Definition

An **operator** is a [linear map](202102071416_linearMapDefinition) from a vector space $V$ to itself.

The set of all operators is denoted $\mathcal{L}(V)$.

Ie $\mathcal{L}(V) := \mathcal{L}(V,V)$
