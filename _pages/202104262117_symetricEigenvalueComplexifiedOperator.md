# Symmetric Eigenvalue Result for a Complexified Operator

If we have a real vector space $V$, $\lambda \in \mathbf{C}$ and $j$ a non-negative integer, then 

$$
(T_C - \lambda I)^j (u + iv) = 0 \iff (T_C - \bar{\lambda} I )^j (u - iv) = 0
$$
