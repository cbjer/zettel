# Permutation Definition

A **permutation** of $(1, \ldots, n)$ is a list $(m_1, \ldots, m_n)$ that contains each of the numbers
$(1, \ldots, n)$ exactly once.

We denote the set of all permutations of $(1, \ldots, n)$ as $\mathrm{perm} \, n$
