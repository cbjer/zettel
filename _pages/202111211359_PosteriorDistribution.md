# Posterior Distribution

See [Bayesian Inference](202101161711_bayesianInference).

$$
\pi(\theta | x) = \frac{f(x|\theta) \pi(\theta)}{\int_{\theta} f(x | \tilde{\theta} ) \pi ( \tilde{\theta} ) d \tilde{\theta}}
$$
