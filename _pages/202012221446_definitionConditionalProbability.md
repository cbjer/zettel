# Definition of Conditional Probability

The definition of conditional probability. It says what is the probability of event A
occurring given event B has occured. 

$$
P (A | B ) = \frac{ P ( A \cap B )}{P(B)}
$$

We also write it as:
$$
P(A, B) = P( A | B ) P(B)
$$
