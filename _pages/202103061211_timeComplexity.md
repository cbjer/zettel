# Time Complexity

Time complexity is the amount of time taken by an algorithm to run as a function of the size 
of the input.

We have:
- Constant time $O(1)$
- Linear time $O(n)$
- Logarithmic time $O(\log n)$
- Quadratic time $O(n^2)$
